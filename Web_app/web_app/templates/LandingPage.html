About this project

This is a culmination of ideas. Originally intended to simply put up a dose per drug calculator, This project is gradually evolving to accomodate the imaginations and challenges some health professionals may face at work. Comparing our state now with our vision for the future, we can clearly say this project is in its embryonic stages. Although this has been drawn out as a project for Holberton School, major updates will definitely roll out soon. Stay tuned
